### Step 2: Set Up Free Database (Choose One)

#### Option A: Firebase (Recommended - Easiest)

1. **Create Firebase project**
   - Go to https://console.firebase.google.com
   - Click "Create Project"
   - Name it "ThePartybox"
   - Click "Create"

2. **Get your credentials**
   - Go to Project Settings
   - Copy the config object
   - Save it

3. **Update backend to use Firebase**
   - Replace SQLite code in `server.js`
   - Use Firebase SDK instead
   - Keep the same API endpoints

#### Option B: Use Vercel + PostgreSQL (Also Free)

Vercel has a free PostgreSQL option:
- Go to Vercel Dashboard
- Add Environment Variables
- Use their free database
